Política de Privacidade

Última atualização: 15 de junho de 2026

A presente Política de Privacidade descreve como o aplicativo Lista de Compras Supermercado coleta, utiliza e protege as informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos desta Política de Privacidade.

1. Informações Gerais

O aplicativo Lista de Compras Supermercado foi desenvolvido para auxiliar usuários na criação e gerenciamento de listas de compras de supermercado de forma simples e prática.

2. Dados Coletados

O aplicativo não exige cadastro, login ou fornecimento de dados pessoais para utilização de suas funcionalidades.

As listas de compras criadas pelo usuário são armazenadas exclusivamente no dispositivo do usuário e não são enviadas para servidores do desenvolvedor.

3. Armazenamento Local

As informações inseridas pelo usuário, como itens da lista de compras, categorias e preferências do aplicativo, são armazenadas localmente no dispositivo e permanecem sob controle do próprio usuário.

O desenvolvedor não possui acesso ao conteúdo das listas criadas no aplicativo.

4. Serviços de Terceiros

O aplicativo pode utilizar serviços fornecidos por terceiros para melhorar a experiência do usuário e garantir o funcionamento adequado do aplicativo.

  • Google AdMob (exibição de anúncios)
  • Google Analytics for Firebase (estatísticas de uso)
  • Firebase Crashlytics (relatórios de falhas e erros)

Esses serviços podem coletar informações técnicas, como modelo do dispositivo, sistema operacional, identificadores de publicidade, idioma, país e dados relacionados ao uso do aplicativo.

Para mais informações, consulte as políticas de privacidade do Google:

  • Política de Privacidade do Google

5. Compartilhamento de Dados

O desenvolvedor não vende, aluga ou compartilha os dados das listas de compras dos usuários com terceiros.

Os dados coletados pelos serviços do Google seguem as respectivas políticas de privacidade desses serviços.

6. Segurança

São adotadas medidas razoáveis para proteger as informações armazenadas pelo aplicativo. No entanto, nenhum método de armazenamento eletrônico é totalmente seguro, e não é possível garantir segurança absoluta.

7. Direitos do Usuário

O usuário pode remover todos os dados armazenados pelo aplicativo a qualquer momento através das configurações do dispositivo ou desinstalando o aplicativo.

8. Alterações nesta Política

Esta Política de Privacidade poderá ser atualizada periodicamente. Recomendamos que os usuários revisem esta página regularmente para acompanhar eventuais alterações.
